CodeIgniter 3 CRUD Assignment

Features:

* Create User
* Update User
* Delete User
* List Users
* State Dropdown from Database
* Form Validation
* SQL Injection Prevention using Query Builder
* XSS Prevention using htmlspecialchars()
* Bootstrap 5 Responsive UI
* Clean URLs using .htaccess

Setup Steps:

1. Import ci_crud.sql into MySQL.
2. Copy the CodeIgniter folder to htdocs.
3. Update database settings in application/config/database.php.
4. Start Apache and MySQL.
5. Open:
   http://localhost/CodeIgniter/user
